# SSR（単純反復配列）検出スクリプト 使用マニュアル

このマニュアルでは、Pythonスクリプト `misa_like_normalized_filtered.2.py` を使用して、ゲノム配列からSSR（Simple Sequence Repeat：マイクロサテライト）を検出する方法を解説します。

## 1. 概要

このスクリプトは、FASTA形式の塩基配列ファイルを入力とし、指定された条件に一致するSSR領域を特定します。

* **対応モチーフサイズ**: 2〜6塩基（デフォルト）
* **特徴**:
* `ATATAT` を `AT` の3回反復として正規化して検出します。
* ホモポリマー（`AAAAA` など）は解析から除外します。
* `.gz` 形式の圧縮ファイルもそのまま読み込めます。



## 2. 準備（Codespaces を使う今回の実習では不要です）

今回の実習環境（GitHub Codespaces）では、解析に必要なツールはすべて自動的にセットアップされます。

* Python 3.11: すでにインストールされています。
* Biopython: ライブラリのインストールも自動的に完了します。
* VS Code 設定: Python 用の便利な拡張機能が最初から有効になっています。

Note: postCreateCommand によって、環境立ち上げ時にバックグラウンドで pip install biopython が実行されます。起動直後はインストールに数十秒かかる場合がありますが、手動でコマンドを打つ必要はありません。

### ライブラリのインストール（Linux, Unixのローカル環境で実行する場合に必要ですが、今回のようにCodespacesを使い場合は不要です）

もし、必要な場合は、ターミナル（またはコマンドプロンプト）で以下のコマンドを実行してください。

```bash
pip install biopython

```

## 3. 実行方法

基本コマンドは以下の通りです。

```bash
python misa_like_normalized_filtered.v2.py [入力FASTAファイル]

```

具体的には、こんな感じ

```
python misa_like_normalized_filtered.v2.py Zea_chr1_region10.fa.gz
```

### オプション設定

必要に応じて、検出するモチーフ（単位配列）の長さを変更できます。

* `--min-unit`: 最小モチーフ長（初期値: 2）
* `--max-unit`: 最大モチーフ長（初期値: 6）

**例：モチーフ長 3〜5塩基のものだけを検出したい場合**

```bash
python misa_like_normalized_filtered.v2.py input.fasta --min-unit 3 --max-unit 5

```

## 4. 出力ファイル

実行が完了すると、配列名（ID）に基づいた2種類のファイルが生成されます。

1. **`.tsv` ファイル**: Excel等で閲覧しやすいタブ区切りテキスト
2. **`.gff` ファイル**: ゲノムブラウザ（IGVなど）で表示可能な注釈ファイル

プレフィックス（拡張子よりも前方部分）として、**"入力FASTAファイル名"_u"最小モチーフ長"to"最大モチーフ長"**　を持つファイルが生成される仕様にスクリプトは設計されています。

具体的には、こんな感じ

```bash
Zea_chr1_region01_u2to6.tsv
Zea_chr1_region01_u2to6.gff

```


### TSVファイルの項目解説

| 項目 | 説明 |
| --- | --- |
| **ID** | 配列の識別子（ヘッダー名） |
| **Start** | SSRの開始位置（1塩基目からカウント） |
| **End** | SSRの終了位置 |
| **Motif** | 繰り返しの単位となる塩基配列 |
| **Repeats** | 繰り返しの回数 |
| **SSR** | 表記形式： (単位配列)回数 |

## 5. 注意事項（学生へのヒント）

* **メモリ**: 非常に巨大な1本（1染色体）の配列を処理する場合、時間がかかることがあります。
* **フィルタリング**: このスクリプトでは、モチーフ長ごとに以下の最小反復回数が設定されています。これ以下の短い反復は検出されません。
* 2塩基：6回以上
* 3〜6塩基：5回以上


* **重複**: モチーフ長が異なるものが重なっている場合、それぞれのサイズで別々に検出されます。
