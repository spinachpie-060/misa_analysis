# misa_analysis/fastas/*.fasta.gz の解説

---

## **1. このデータは何か？**

```bash
misa_analysis/fastas/*.fasta.gz
```

に含まれるファイルは、

👉 **トウモロコシ（Zea mays）リファレンスゲノムから抽出したDNA配列（FASTA形式）**

です。

👉 **第1染色体から第10染色体のそれぞれの10箇所から**抜き出した**1 Mb（1,000,000 bp）**のDNA断片です。

---

## **2. 元データについて**

元となる配列は：

- トウモロコシ B73 参照ゲノム
- 染色体ごとに数千万〜数億塩基の長さ

👉 非常に大きなデータです

---

## **3. このデータの作成ルール（最重要）**

これらの配列は、

👉 **単純に端から1 Mbずつ分割したものではありません**

👉「長いDNAを切り分けたものではなく、ゲノム全体からバランスよく抜き出した1 MbのDNA断片です」

---

### **✔ 実際のルール**

各染色体について：

- **1 Mb（1,000,000 bp）の配列を**
- **染色体全体から均等に10カ所抽出**

---

### **✔ 具体的な仕組み**

1. 染色体の長さを取得（.faiファイル）
2. 以下の式で間隔を計算

```
step = (染色体長 − 1,000,000) / (10 − 1)
```

1. 等間隔で開始位置を決定

```
start = 1, 1+step, 1+2×step, ...
```

1. 各位置から **1 Mb の領域を切り出し**

---

## **4. イメージ**

```
[ 染色体 ]
|----|----|----|----|----|----|----|----|----|----|
^    ^    ^    ^    ^    ^    ^    ^    ^    ^
抽出位置（10カ所）
```

👉 **染色体全体にバランスよく配置される**

---

## **5. ファイル形式**

### **✔ FASTA形式**

```
>Chr01_pos01
ATGCGT...
```

- `>`：配列名
- 下：塩基配列

---

### **✔ gzip圧縮**

```bash
*.fasta.gz
```

👉 容量削減のため圧縮

### **✔ gzip圧縮ファイルの内容確認方法（zcat、 パイプ、less を活用しよう）**

```bash
ls -1 ./fastas/*.fasta.gz #ディレクトリ fastas 内の *.fasta.gz が1列でリストアップされる
ls -1 ./fastas/*.gz | head #ディレクトリ fastas 内の *.fasta.gz が10個だけ表示される。"ls -1" による出力が、"パイプ" で "head"　に渡されて処理される。
ls -1 ./fastas/*.gz | less #ディレクトリ fastas 内の *.fasta.gz がスクロール表示される。"ls -1" による出力が、"パイプ" で "less"　に渡されて処理される。
ls -1 ./fastas/*.gz | wc -l #ディレクトリ fastas 内の *.fasta.gz のファイル数が表示される。"ls -1"  による出力が、"パイプ" で "wc -l"　に渡されて処理される。
zcat ./fastas/Chr01_pos01.fasta.gz | less #ディレクトリ fastas 内の Chr01_pos01.fasta.gz がスクロール表示される。"zcat" による出力が、"パイプ" で "less" に渡されて処理される。
```
