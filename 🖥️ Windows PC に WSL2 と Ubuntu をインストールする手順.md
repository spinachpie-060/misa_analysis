# 🖥️ Windows PC に WSL2 と Ubuntu をインストールする手順

---

# 🐧 Ubuntuとは？

**Ubuntu（ウブントゥ）は、Linux（リナックス）というOSの中でも、特に初心者にやさしく使いやすい**ことで知られる人気ディストリビューション（配布形態）です。

### ✅ 特徴：

- **無料で使えるオープンソースOS**
- WindowsやMacの代替として使える
- ソフトウェアのインストールも簡単（`apt`コマンドなど）
- デスクトップ版・サーバー版の両方あり
- WSL（Windows Subsystem for Linux）でWindowsでも利用可能

### 💡 Ubuntuを選ぶ理由：

- 初心者でも扱いやすいGUI（デスクトップ画面）あり
- ドライバや日本語入力などの設定がしやすい
- コミュニティが活発で、困ったときにネットで解決しやすい

### 🧪 よく使われる場面：

- プログラミング学習
- サーバー運用
- バイオインフォマティクスや研究開発
- 古いPCの再活用

# **WSL2 と Ubuntu をインストールする手順**

---

**✅ Step 1: WSL2 の有効化**

**1.1 Windows のバージョン確認**

WSL2を利用するには、以下のOSが必要です：

- Windows 10（ビルド1903以降）
- または Windows 11

**バージョンの確認方法：**

1. Win + R を押す

2. winver と入力 → Enter

3. バージョンが **1903以上**であることを確認

---

**1.2 WSLの有効化と Ubuntu のインストール**

以下の手順でWSL2とUbuntuをインストールします。

**手順：**

1. **「PowerShell」または「Windows Terminal」** を **管理者として実行**

2. 以下のコマンドを入力して実行：

```
wsl --install
```

これで WSL2 が有効化され、Ubuntu（最新版）が自動的にインストールされます。

途中で再起動を求められたら、PCを再起動してください。

---

**1.3 WSL のバージョンを確認・変更（必要な場合のみ）**

WSLの状態を確認するには：

```
wsl --list --verbose
```

UbuntuがWSL1になっている場合は、次のようにしてWSL2へ変更できます：

```
wsl --set-version Ubuntu 2
```

---

**✅ Step 2: Ubuntu を起動してセットアップ**

**2.1 Ubuntu の起動方法（初回起動）**

**方法①（おすすめ）：PowerShell または Windows Terminal から起動**

```
wsl
```

**方法②：スタートメニューから起動**

1. 「スタートメニュー」→「Ubuntu」と検索

2. 表示されたアイコンをクリック

---

**2.2 ユーザー名とパスワードの設定**

初回起動時に、以下のように聞かれます：

- Unix username（好きな名前）
- パスワード（2回）

※ これは Ubuntu 内で使うログイン用の設定です。Windowsとは別のものです。

---

**2.3 パッケージの更新（初回だけ）**

Ubuntuが起動したら、次のコマンドを実行して最新状態にしておきます：

```
sudo apt update && sudo apt upgrade -y
```

- sudo：管理者としてコマンドを実行
- apt：パッケージ管理ツール

# Visual Studio Code のインストールと設定

---

## 0. 🧑‍💻 Visual Studio Code（VSCode）とは？

**Visual Studio Code（通称：VSCode）は、Microsoft が開発した無料のコードエディタ**です。軽量で動作が速く、拡張機能が非常に豊富なため、世界中の開発者や研究者に使われています。

### ✅ 主な特徴

- **無料で使える & 軽量**
- さまざまな言語（Python, C++, HTML など）に対応
- 拡張機能を入れることで、機能を自由に追加できる
- **Git連携**や**ターミナル内蔵**など、開発に便利な機能が多数
- WSL（Windows Subsystem for Linux）との相性も抜群！

### 💡 初心者にもオススメの理由

- 日本語化が簡単
- シンプルな見た目で直感的に操作できる
- Linux 環境（Ubuntuなど）にもスムーズに接続できる
- 自分に合った使い方ができる（コード補完、色テーマなど）

---

## 1. Visual Studio Code のインストール

### 1.1. ダウンロードとインストール

1. windowsストアでVisual Studio Code のインストール

### 1.2. 日本語設定

1. **VSCode** を起動。
2. 機能拡張ボタンをクリック。
3. `Japanese Language Pack for Visual Studio Code` を検索し、インストール。
4. VSCode を再起動すると日本語 UI になる。

## 2. Remote developmenの導入

### 2.1. Remote - WSL 拡張のインストール

1. 「拡張機能」を選択（ショットーカットキー：Ctrl＋Shift＋X）
2. 上部テキストボックスに「remote」と入力
3. 表示されたリストから「Remote Development」を選び「インストール」をクリック

### 2.2. Ubuntu への接続

1. VSCodeの左下の隅に，"><"のようなボタンがありますので，これをクリック
2. 「WSLへの接続」を選択
3. 終了時は「リモート接続を終了する」を選択

---

# まとめ

1. **WSL2 を有効化** し、Ubuntu をインストール。
2. **Ubuntu をセットアップ**（ユーザー作成、アップデート）。
3. **VSCode をインストール** し、日本語化。
4. **機能拡張** Remote development を導入し、VSCode から Ubuntu を操作。

この手順に従えば、Windows 上で快適に Ubuntu 環境を構築し、VSCode で開発できるようになります。
